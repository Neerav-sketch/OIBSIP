# Unemployment Analysis with Python

## Objective

Analyze unemployment trends in India using Python and visualize insights through Exploratory Data Analysis (EDA).

## Dataset

- Dataset: Unemployment in India
- Source: Kaggle

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Steps Performed

1. Imported the dataset
2. Cleaned missing values
3. Converted date column
4. Performed Exploratory Data Analysis (EDA)
5. Visualized unemployment trends
6. Compared unemployment before and during COVID-19
7. Generated insights

## Outcome

Successfully analyzed unemployment trends across different states and time periods using data visualization.
