# Car Price Prediction with Machine Learning

## Objective

Develop a machine learning model to predict the selling price of used cars based on features such as manufacturing year, fuel type, transmission, kilometers driven, owner history, and present price.

## Dataset

- Dataset: Car Price Prediction (Used Cars)
- Source: Kaggle
- Link: https://www.kaggle.com/datasets/vijayaadithyanvg/car-price-predictionused-cars

## Tools Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib

## Steps Performed

1. Imported the dataset
2. Explored and cleaned the data
3. Checked for missing values
4. Encoded categorical variables
5. Selected important features
6. Split the dataset into training and testing sets
7. Trained a Linear Regression model
8. Evaluated model performance using R² Score, MAE, MSE, and RMSE
9. Predicted car selling prices
10. Saved the trained model

## Outcome

Successfully built a machine learning model capable of predicting the selling price of used cars with good accuracy.
